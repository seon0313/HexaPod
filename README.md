## 개요
서울로봇고등학교 *--졸업작품 or 캡스톤* 출품 6족 거미 로봇.

정찰용 드론이 머리위에 있으며 필요에 따라 로봇팔, 짐벌등 확장 모듈을 머리 위에 설치 할 수 있다.
